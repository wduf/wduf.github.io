Interactive Data Structure Visualizer (Link)

Made with:

I built an interactive tool for visualizing different data structures. The user can add or remove values from each data structure, which is then displayed using the Canvas API. Currently, the tool works for LinkedLists, Stacks, Queues, Binary Search Trees, and Max Heaps, but I plan on adding more data structures like Graphs and Maps in the future.

Connect Four with Smart Bot

Made with:

I created a version of Connect Four that runs in the Windows terminal and gives the option to play against another person or a smart bot. For every legal move, the bot checks in all seven directions and calculates a score for each column based off of whether it sees a streak in that and the opposite direction, an opportunity to block the opponent, or a win. It then selects the column with the highest score or picks a random one if there is a tie.

NFL Combine Draft Predictor

Made with:

Using an NFL Combine dataset from Kaggle, a group of three classmates and I created several machine learning models and compared them to see which could most accurately predict what round players would get drafted based off of their performance in the Combine. We took multiple different approaches, including linear regression, random forest regression, and k-nearest neighbors, all combined with principal component analysis and a grid search to find the optimal hyperparameters .
